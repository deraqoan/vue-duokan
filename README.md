# 多看阅读


## 本地运行 

使用vue-cli工具构建，基本命令如下：
``` bash
# 安装依赖
npm install 

# 开发模式
npm run dev

# 生产模式
npm run build

```
## 预览地址
项目放在google云的虚拟主机上，访问速度看人品。代理也运行在上面，所以加载速度可能会比较慢。其中有一些小说封面会加载不出来，这个是api的问题，并不是网络的原因。


电脑端请开启开发者模式
